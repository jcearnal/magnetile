Layouts that work across KWin scripts

Magnetile, KZones, and PlasmaZones use compatible layout JSON with pixel padding and percentage-based zones. Optional zone metadata such as applications, indicators, and colors is preserved when you import existing JSON.

Optional Magnetile-only trigger area for this zone.

Use this when a zone should be selected from a screen edge that is different from where the zone places windows. For example, a fullscreen zone can use the top edge while left and right split zones use the side edges. Start and size describe the active strip along that edge: horizontal for top/bottom, vertical for left/right. Leave every edge unchecked to use the normal Magnetile behavior.
